# Удалить Docker-тесты

## Проблема

В проекте есть отдельный набор тестов (`testsDocker/`), которые запускаются внутри Docker-контейнера через скрипт `test-docker.sh`. Они проверяют три вещи: VAPID-ключ отдаётся, уведомление принимается, CORS работает.

Все три проверки полностью дублируются e2e тестами в `tests/e2e/app.test.ts`, которые при этом лучше: изолированы, не требуют Docker, покрывают больше кейсов.

## Желаемое поведение

Docker-тесты полностью удалены из проекта. `Dockerfile` и `docker-compose.yml` остаются — они нужны для деплоя.
