## ✅ 1. Создать модуль загрузки окружения

**Файл:** `src/env.ts`

Фабричная функция `loadEnv()`:
- Вызывает `process.loadEnvFile('.env')` в try/catch
- Если файл не найден — генерирует VAPID-ключи через `webpush.generateVAPIDKeys()`, формирует содержимое `.env` с дефолтами (`PORT=3001`, `VAPID_SUBJECT=mailto:test@example.com`), записывает файл, вызывает `process.loadEnvFile('.env')` повторно
- Выводит в консоль предупреждение при генерации новых ключей
- Возвращает объект с прочитанными переменными (`port`, `vapidSubject`, `vapidPublicKey`, `vapidPrivateKey`)

- Удалить скрипт `setup`
